* ## [Temporary Host](https://guarded-headland-61604.herokuapp.com/)
* ## Usage
  * **`npm install`** <br>
    install npm package (if you haven't installed it before)
  * **`cd <project folder>`** <br>
    direct to the project folder
  * **`npm start`** <br>
    start npm package
  * **`npm run build`** <br>
    build the app for production if you want
